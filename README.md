# ZenCloud Advisory

Public website source for ZenCloud Advisory.

Live site: https://www.zencloud.com.au/

React + Vite single-page site for ZenCloud Advisory (enterprise architecture and cloud advisory).

## Development

```bash
npm ci
npm run dev
```

Production validation:

```bash
npm run build
npm run lint
```

## Deployment

The `main` branch is built with `npm run build`; the generated `dist` directory is published to `www.zencloud.com.au`.
